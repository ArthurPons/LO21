\subsection{Detailed Description}
Classe représentant le manager de \hyperlink{class_programmation}{Programmation}. 

Le manager de \hyperlink{class_programmation}{Programmation} est unique et ne doit être instancié qu\textquotesingle{}une seule fois. Il utilise donc le design pattern singleton. Il y a une composition entre \hyperlink{class_programmation_manager}{Programmation\+Manager} et \hyperlink{class_programmation}{Programmation}. En effet, \hyperlink{class_programmation_manager}{Programmation\+Manager} détermine la durée de vie d\textquotesingle{}un \hyperlink{class_programmation}{Programmation}.
